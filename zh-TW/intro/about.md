# 關於 Microlattice.js

## 命名精神

* 中文名：微晶格
* 英文意思：世界上最小最輕且非常堅固的金屬材料


## 設計理念
* 針對 Javascript community 所熟悉的 coding style 追求接近 C 的效能
* 講求輕量化
* 每一個細節包含 tool, module, engine core 都是可以拆分重組
