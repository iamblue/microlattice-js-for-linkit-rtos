# 如何創建 Microlattice.js 專案?


## 步驟
* 請先具備好 Node.js (> 0.10.32 以上)環境
* npm install microlattice -g
* 創建一個新的資料夾並進去: mkdir testSDK && cd testSDK
* ml create
* 編輯 featureConfig.json:
``` bash
{
  "IC_CONFIG": "mt7687",
  "BOARD_CONFIG": "mt7687_hdk",
}
```
* npm install ml-mt7687-config --save
* ml init:mt7687
* npm i
